Server transmite el paquete .msi al cliente

2 — Adquisicion

msiexec lee tablas OLE-DB en memoria (SQL COM)

3 — Elevacion UAC

Solicitud de token elevado para modo SYSTEM

4 — Delegacion

Control pasa a TrustedInstaller (servicio SYSTEM)

5 — Ejecucion

Escritura en Program Files, Registry y Local DB
