ETAPA 1 / 4 — RECOVERY POINT OBJECTIVE (RPO)

¿Qué tanto estoy dispuesto a perder?

MTD=RTO+WRT
RESUMEN CONCEPTUAL

RPO = tolerancia a pérdida de datos antes del desastre. RTO = tiempo hasta tener el sistema recuperado. WRT = tiempo adicional para validar integridad y reanudar producción. MTD = ventana total tolerable de inactividad = RTO + WRT.
